HTML 与图片

HTML → 图片：将片段渲染为 PNG（简单布局效果较好）；图片 → HTML：生成带内嵌 data URL 的 <img> 片段。

SVG 与图片

SVG → 栅格图：导出 PNG；图片 → SVG：生成包裹位图的 SVG（矢量追踪需专业工具）。
